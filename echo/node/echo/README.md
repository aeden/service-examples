# echo

Web service written in JavaScript (to run on Node.js) that parses and echoes JSON data.

## Usage

Install node and then run the service:

    node app.js

Call the service from curl:

    curl -i "http://127.0.0.1:3000" -d '{"foo":"bar"}'
